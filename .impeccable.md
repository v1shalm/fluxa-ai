# Fluxa — Design Context

This file is the source of truth for design direction. Synthesized from working
sessions; update in place when intent shifts.

## Product

**Fluxa** — a visual canvas + typed runtime for shipping LLM workflows to
production. Wire LLMs, tools, and data on a typed canvas; deploy as an
observable API; trace every run, every token. Self-hostable.

## Users

Backend / AI / ML engineers shipping LLM-powered products. Already comfortable
with Postgres, Vercel, Linear, GitHub, Datadog. They reach for tools that respect
their time and their typing — not tools that explain themselves with marketing
fluff. They distrust anything that smells like "AI-ifying" software.

**Job-to-be-done**: prove (in one scroll) that Fluxa is *the* serious runtime
for production LLM workflows — not another notebook, not another no-code
playground, not another agent SaaS.

## Brand personality

Three words: **typed, instrumented, opinionated.**

- *Typed* → the artefact is real code; nothing magical.
- *Instrumented* → every node, every token, every run is observable.
- *Opinionated* → there is one obvious way; we're not a Lego box.

Think: Linear's quietness, Vercel's clarity, oxide.computer's industrial
restraint, Rauno Freiberg's craft attention. **Not** Notion AI, **not** any
"AI infrastructure platform" with cyan→purple gradients on a starry hero.

## Aesthetic direction

- **Theme**: dark. Audience uses tools at night; brand colors only sing on dark.
- **Surface**: a single tinted ink (`#0A0A0A` → `#161616`), tight grid, grain.
  No layered cyan + purple + green radial washes — that combination is the
  single most recognizable AI-marketing-site fingerprint of 2024-2025.
- **Color use**: 60-30-10. The six flux accents
  (cyan / green / lime / yellow / pink / purple) are the *spectrum motif* — they
  belong almost exclusively to (a) the hero pipeline cards, (b) the tiny brand
  mark, and (c) one signature horizontal bar in the footer. **Everywhere else,
  one accent at a time, used sparingly.** Random rainbow icon tiles in nav
  menus and feature cards are banned.
- **Typography**: Funnel Display (display) + Funnel Sans (body). Already
  loaded. Big size contrast (~1.25 ratio), tight tracking on display
  (`-0.035em` to `-0.04em`), monospace reserved for code, version chips,
  and hero anchor labels (INPUT / OUTPUT).
- **Motion**: ease-out-expo / ease-out-quart. Long, decisive entrances.
  No bounce, no spring. The hero pipeline gets one premium choreography
  (line draws → comet threads cards → erases) and that motif sets the
  pace for everything else.

## Anti-patterns (banned)

- Gradient text (any `background-clip: text` + gradient).
- Side-stripe borders (`border-left: Npx solid <color>` where N > 1) on
  cards, list items, callouts.
- Colored-tile icon containers (`bg-flux-cyan/15` square wrapping an icon).
- Generic accent drop-shadow on every card (`box-shadow: 0 0 40px <accent>`).
- Stacked AI-palette atmospheres: a single page-level wash with cyan +
  purple + green at the top is on every Notion AI / Vercel-AI clone.
- Generic SaaS arrow → on every primary button.
- Loading-bar-as-decoration in nav menus.
- Hero stat triplets ("8.2k teams · 412k workflows · 2.4M runs / day") with
  every number animated. Show one number that means something, or none.

## Design principles (decision-makers)

1. **Show, don't decorate.** A motion choice must say something about the
   product (data flowing through typed nodes, runs being traced). Decorative
   motion that doesn't reinforce the runtime metaphor gets cut.
2. **Color is currency.** Every accent costs attention. The hero spectrum
   is paid; everything else is one accent or none.
3. **Code-shaped affordances.** Lowercase tags, monospace for telemetry,
   tabular numerals, filename-style metadata strips. Avoid pill-soup.
4. **One signature per section.** Each section earns its place by doing one
   distinct thing — not by stacking another card grid + another headline +
   another CTA.

## Things explicitly OK

- The hero pipeline cards (rainbow cards) — they're the brand's spectrum
  motif. They get to be saturated.
- The footer's giant `fluxa` wordmark with the scrolling rainbow bar
  underneath — signature moment, leave it alone.
- The grid + grain page background — tasteful, restrained.
